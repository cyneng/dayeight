CSS Syntax

The format of css is different than html. An example is listed below.

selector {
      property: value;
}

A selector can be any HTML element such as an img, table, or an h1.

A property is an aspect of a selector. A value is a setting for a property. So a property can be font-family, color, or font-size. A value can be blue, red, Verdana, Arial, etc.

In order for the CSS to know that you are done with a property-value, you need to end each pair with a semi-colon.

Font Size

When specifying font sizes, you can do so in px (pixels) or ems. A pixel is a dot on the computer screen. Use px when assuming the screens of whoever is viewing your site is going to be of similar size. So that the user sees exactly what you see. IF a user is using a different size, use ems.

An em is a relative measure. One em is equal to th default font size on whatever screen the user is viewing the site on. For a smartphone, 1em is the fontsize you normally use, .5em is half the size.
